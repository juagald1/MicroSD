# MicroSD
Firmware para gestión de tarjeta Micro SD con microcontrolador (STM32+SDIO+FATFS)


# Hardware Requerido
- Microcontrolador STM32 con periférico SDIO disponible
- Holder SD (Montaje) o similar
- Lector tarjetas microSD

# Software Requerido
- STM32CubeIDE
